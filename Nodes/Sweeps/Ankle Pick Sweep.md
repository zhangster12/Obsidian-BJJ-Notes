# Overview
The <u>Ankle Pick Sweep</u> is where you grab the passer’s ankle and use it to sweep from guard.
# Setup
- [Deep DLR-X Guard](obsidian://open?vault=Obsidian-BJJ-Notes&file=Guards%2FDeep%20DLR-X%20Guard): If the passer kneels in their far leg, pull their ankle towards you.
- Dogfight:
- Shin-on-Shin Guard:
# Keys
- **Ankle/toe control**: If the passer is standing or on their toes, control the ankle; if they’re not, control the toes.
